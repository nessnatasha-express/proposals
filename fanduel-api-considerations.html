üîÑ Critical Data Flow Patterns

Real-time Personalization Flow

1
User action triggers event in Databricks
2
KNIME model scores user for promo eligibility
3
ML Promo Engine calculates optimal offer
4
Hightouch syncs decision to Braze
5
Braze triggers campaign with Movable Ink
6
Movable Ink renders personalized visual

Content Delivery Pipeline

1
Marketing creates content in Sanity CMS
2
Assets stored and approved in Bynder
3
Braze Connected Content calls Sanity API
4
Movable Ink pulls assets from Bynder API
5
Dynamic content rendered at open time
6
Engagement data flows back to Databricks

Risk Management Flow

1
Betting events stream to Databricks
2
KNIME fraud models analyze patterns
3
Risk scores updated in real-time
4
Hightouch syncs alerts to operational systems
5
ML Promo Engine adjusts limits automatically
6
Compliance notifications sent via Braze
